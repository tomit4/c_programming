Give the decimal value of each of the following integer constants.

(a) 077
(b) 0x77
(c) 0XABC

A:
077
(a) Octal.

```
7 * 8^1 + 7 * 8^0
(7 * 8) + (7 * 1) = 56 + 7 = 63
A: 63
```

(b) Hexadecimal

```
7 * 16^1 + 7 * 16^0
(7 * 16) + (7 * 1) = 112 + 7 = 119
A: 119
```

(c) Hexadecimal

```
10 * 16^2 + 11 * 16^1 + 12 * 16^0
(10 * (16 * 16)) + (11 * (16)) + (12 * 1) = 2748
A: 2748
```
