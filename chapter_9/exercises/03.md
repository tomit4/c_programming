Q: Write a function `gcd(m, n)` that calculates the greatest common divisor of
the integers `m` and `n`. (Programming Project 2 in Chapter 6 describe's
Euclid's algorithm for computing the GCD).

A:

```c
int gcd(int m, int n)
{
    int remainder;

    while (n != 0) {
        remainder = m % n;
        m = n;
        n = remainder;
    }

    return m;
}
```
