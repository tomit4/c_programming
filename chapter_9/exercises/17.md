Q: Rewrite the `fact` function so that it's no longer recursive.

```c
int fact(int n)
{
    if (n <= 1) {
        return 1;
    } else {
        return n * fact(n - 1);
    }
}
```

A:

```c
int fact(int n)
{
    int factorial = 1;

    while (n > 1) {
        factorial *= n;
        n--;
    }
    return factorial;
}
```
