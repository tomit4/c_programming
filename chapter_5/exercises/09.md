Q: Are the following `if` statements equivalent? If not, why not?

```c
// First `if` statement
if (score >= 90) {
    printf("A");
} else if (score >= 80) {
    printf("B");
} else if (score >= 70) {
    printf("C");
} else if (score >= 60) {
    printf("D");
} else {
    printf("F");
}
```

```c
// Second `if` statement
if (score < 60) {
    printf("F");
} else if (score < 70) {
    printf("D");
} else if (score < 80) {
    printf("C");
} else if (score < 90) {
    printf("B");
} else {
    printf("A");
}
```

A: Yes, they are equivalent.
