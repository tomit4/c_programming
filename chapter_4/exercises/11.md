Q: Show the output produced by each of the following program fragments. Assume that
`i`, `j`, and `k` are `int` variables.

(a

```c
i = 1;
printf("%d ", i++ - 1);
printf("%d", i);
```

(b)

```c
i = 10;
j = 5;
printf("%d ", i++ - ++j);
printf("%d %d", i, j);
```

(c)

```c
i = 7;
j = 8;
printf("%d ", i++ - --j);
printf("%d %d", i, j);
```

(d)

```c
i = 3;
j = 4;
k = 5;
printf("%d ", i++ - j++ + --k);
printf("%d %d %d", i, j, k);
```

A:
a: 0
a: 2
b: 4
b: 11 6
c: 0
c: 8 7
d: 3
d: 4 5 4
