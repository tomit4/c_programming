Q: Show the output produced by each of the following program fragments. Assume that
`i` and `j` are `int` variables.

(a)

```c
i = 5;
j = ++i * 3 - 2;
printf("%d %d", i, j);
```

(b)

```c
i = 5;
j = 3 - 2 * i++;
printf("%d %d", i, j);
```

(c)

```c
i = 7;
j = 3 * i-- + 2;
printf("%d %d", i, j);
```

(d)

```c
i = 7;
j = 3 + --i * 2;
printf("%d %d", i, j);
```

A:

a: 6 16
b: 6 -7
c: 6 23
d: 6 15
